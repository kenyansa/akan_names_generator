# age_calculator
### version 1.0.0
This app generates for you an Akan name, based on  your week day of birth 

## Contributors
The project was developed by Justine Kenyansa.

## :flashlight: Description
The user should select their gender and DOB. <br>
I first came across such an assignment in Moringa School, a year ago. Wanted to try it again.
## Male Names:

Sunday: Kwasi (Kwesi)
Monday: Kwadwo (Kojo)
Tuesday: Kwabena
Wednesday: Kwaku
Thursday: Yaw
Friday: Kofi
Saturday: Kwame
## Female Names:

Sunday: Akosua
Monday: Adwoa (Abena)
Tuesday: Abena (Adwoa)
Wednesday: Akua
Thursday: Yaa
Friday: Afua (Afuomaa)
Saturday: Ama
